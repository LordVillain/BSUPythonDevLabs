# Анализ кредитных рисков (German Credit Data)

Данный проект представляет собой полный цикл анализа данных (ETL, EDA, SQL) на основе набора данных **German Credit Data** из репозитория UCI Machine Learning.

**Цель работы:** Исследование факторов, влияющих на кредитоспособность клиентов, визуализация ключевых метрик и организация хранения данных в реляционной базе данных (SQLite).

## Структура проекта

Проект организован по модульному принципу:

```text
german_credit_project/
│
├── data/
│   └── raw/
│       └── german.data        # Исходный датасет
│
├── src/                       # Исходный код модулей
│   ├── config.py              # Конфигурация, пути и словари расшифровки
│   ├── processing.py          # Загрузка, очистка и обработка данных
│   ├── visualization.py       # Генерация графиков и диаграмм
│   └── database.py            # Экспорт в SQLite и выполнение запросов
│
├── output/                    # Папка для результатов (создается автоматически)
│   ├── figures/               # Сохраненные графики (.png)
│   └── database/              # Файл базы данных (.db)
│
├── main.py                    # Главный скрипт запуска
├── requirements.txt           # Список зависимостей
└── README.md                  # Описание проекта
```

## Функциональность

1.  **Загрузка данных:** Чтение исходного файла `.data`, присвоение имен столбцов согласно спецификации UCI.
2.  **Предобработка:**
    *   Проверка на пропуски.
    *   Декодирование категориальных признаков (например, `A11` -> `< 0 DM`, `A34` -> `critical account`).
    *   Преобразование целевой переменной в читаемый формат (`Good`/`Bad`).
3.  **Визуализация:**
    *   Гистограммы распределения возраста.
    *   Boxplot зависимости суммы кредита от риска.
    *   Анализ целей кредитования.
    *   Тепловая карта корреляции (Heatmap).
4.  **База данных (SQL):**
    *   Автоматическое создание БД SQLite.
    *   Экспорт обработанного датафрейма в таблицу `clients`.
    *   Выполнение аналитических SQL-запросов (агрегация, группировка, выборка).

## Установка и настройка

Для работы проекта требуется **Python 3.8+**.

1.  **Клонируйте репозиторий** или скачайте архив с проектом.
2.  **Убедитесь, что исходный файл на месте:**
    Файл `german.data` должен находиться в папке `data/raw/`.
3.  **Установите необходимые библиотеки:**

    Вы можете установить их вручную:
    ```bash
    pip install pandas matplotlib seaborn scikit-learn
    ```
    Или используя файл зависимостей (создайте файл `requirements.txt` с содержимым ниже):
    ```bash
    pip install -r requirements.txt
    ```

## Запуск

Находясь в корневой папке проекта, выполните команду:

```bash
python main.py
```

## Результаты работы

После успешного выполнения скрипта:

1.  В консоли отобразится статистическая сводка и результаты SQL-запросов.
2.  В папке `output/figures/` появятся 4 графика:
    *   `1_age_distribution.png`
    *   `2_amount_boxplot.png`
