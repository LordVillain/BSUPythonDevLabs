
# University Schedule Bot

Телеграм-бот для просмотра расписания занятий. Бот умеет парсить сложные Excel-таблицы (Google Sheets) с объединенными ячейками, поддерживает несколько курсов и поиск по преподавателям.

## Возможности

*   **Мульти-курс:** Можно добавить несколько ссылок на разные таблицы (1-4 курсы).
*   **Умный парсинг:**
    *   Автоматически находит структуру таблицы (потоки, группы, кафедры).
    *   Корректно обрабатывает объединенные ячейки.
    *   Фильтрует "мусор".
    *   Распознает потоковые лекции и разделение по подгруппам.
*   **Поиск расписания преподавателя:**
    *   Поиск по фамилии по всем добавленным курсам.
    *   Вывод расписания с указанием групп и аудиторий.
*   **Красивый вывод:** Структурированные сообщения с эмодзи.

## Установка и запуск

### 1. Клонирование репозитория
Скачайте проект или склонируйте его через Git:
```bash
git clone https://github.com/LordVillain/BSUPythonDevLabs
cd Parser
```

### 2. Установка зависимостей
Установите необходимые библиотеки из файла requirements.txt:
```bash
pip install -r requirements.txt
```

### 3. Настройка
1. Откройте файл `config.py`.
2. Вставьте свой токен от бота (получить у @BotFather):
   ```python
   BOT_TOKEN = "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
   ```

### 4. Запуск
Запустите бота командой:
```bash
python main.py
```
